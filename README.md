## What's Newの更新について
- `_posts`フォルダの中に`YYYY-MM-DD-title.md`といったmarkdown文書を入れるとブログ記事としてポストされます。
- 投稿日はファイル名から自動的に生成されます。
- title部分が記事のリストとして表示されます。
- 過去の投稿をテンプレートとしてコピーして、編集してください。
